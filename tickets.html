📋 Google Forms Integration

To add your Google Form:

  1. Create a Google Form at forms.google.com
  2. Click "Send" → Click the embed icon (<>)
  3. Copy the iframe code
  4. Replace the placeholder in tickets.html

Example: 

<iframe 
    src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true" 
    width="100%" 
    height="500" 
    frameborder="0" 
    marginheight="0" 
    marginwidth="0">
    Loading…
</iframe>

💰 Stripe Payment Integration

To add your Stripe payment page:

  1. Log in to Stripe Dashboard
  2. Go to Products → Add Product
  3. Create a Payment Link
  4. Copy the payment link
  5. Replace the placeholder in tickets.html

Example: 

<iframe 
    src="https://buy.stripe.com/test_XXXXXXXX" 
    width="100%" 
    height="400" 
    frameborder="0">
    Loading…
</iframe>
